# wtwd-app-h5

- webpack4多页应用

> 依赖
- webpack@4
- highcharts
- art-template
- typescript 
- less

## install

```
$ cnpm i
```

## dev

```bash
$ npm start

# 立方体       => http://localhost:8080/cube.html
# 立方体3d全景  => http://localhost:8080/cube3d.html
# 桃心线方程    => http://localhost:8080/heart.html
# 造物节全景    => http://localhost:8080/xinlv.html
```

## build

```
$ npm run build
```
